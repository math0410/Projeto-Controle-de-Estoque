📦 Sistema de Controle de Estoque

Este projeto é uma aplicação web de controle de estoque, desenvolvida utilizando HTML, CSS, JavaScript e MySQL. O sistema permite gerenciar produtos, acompanhar quantidades e organizar entradas e saídas de forma simples e eficiente.

📌 Funcionalidades
📋 Cadastro de produtos
✏️ Edição e exclusão de itens
📦 Controle de quantidade em estoque
⬆️ Registro de entradas de produtos
⬇️ Registro de saídas de produtos
🔍 Busca de produtos por nome
📊 Visualização organizada dos dados
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla)
MySQL (Banco de Dados)
